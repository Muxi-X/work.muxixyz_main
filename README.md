## 关于分页/PAGE

+ 返回数据格式/RETURN_DATA_STRUCTURE

```
{ 
    "count": integer,   //总数据量
    "pageMax": integer, //总页数
    "pageNow": integer, //当前页
    "hasNext": bool,    //是否有下一页
    "list": list(),     //数据列表
}
```

大多数参数在```page.py```中的```get_rows()```的返回值中。
